# lab-cleaner-checker

lab-cleaner-checker check Friday cleaner

## Requirements

- Ruby 2.5.0

## Setup

```
$ bundle install
```

## Usage

```
$ bundle exec ruby lab-cleaner-checker.rb
```

### Environment Values

- `SLACK_TOKEN` (required)
- `SLACK_CHANNEL`
- `REDIS_URL` (required)
- `NUM_MAX_GROUP`
